Este proyecto implementa un sistema sencillo de búsqueda de rutas dentro del sistema TransMilenio utilizando el algoritmo A*. El objetivo es permitir que el usuario ingrese una estación de inicio y una de destino, y el programa calcule la ruta más corta basada en una base de conocimiento construida para el ejercicio. El desarrollo se realizó en Google Colab, donde se ejecutó y probó el código. Se utilizaron las siguientes tecnologias Python Google Colab Algoritmo de busqueda A Estructura de grafos Github como repositorio del proyecto El codigo se ejecuta de la siguiente manera Abrir el archivo .ipynb en Google Colab.

Ejecutar todas las celdas.

Estación de inicio

Estación de destino
